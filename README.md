## 🔧 Build Instructions

### 📋 **Prerequisites**

- **Operating System**: Windows 10/11 (Administrator privileges required)
- **Compiler**: Visual Studio Build Tools 2022 or newer
- **Assembler**: NASM (Netwide Assembler) for low-level routines
- **Additional Tools**: Rust/Go toolchain for auxiliary modules
- **Drivers**: WinDivert/npcap for network interception
- **Hardware**: TPM 2.0 chip (recommended for hardware security)

### 🚀 **Quick Build**

```powershell
# Clone and navigate to project
git clone https://github.com/LackadaisicalSecurity/LACKYVPN.git
cd LACKYVPN

# Build all components (Debug mode)
make all

# Build optimized release version
make release

# Build stealth version (anti-analysis)
make stealth

# Run comprehensive test suite
make test

# Package for deployment
make package
```

### 🔧 **Advanced Build Options**

```powershell
# Build specific components
make crypto          # Cryptographic library only
make drivers         # Kernel drivers only
make ui              # Electron interface only

# Development builds
make debug           # Debug symbols enabled
make profile         # Performance profiling enabled
make sanitize        # Memory sanitizer enabled

# Specialized builds
make hardened        # Maximum security hardening
make portable        # Portable executable
make minimal         # Minimal feature set
```
